## Prerequisites

<Callout kind="info" title="Before You Begin">
To complete this quickstart, ensure you have:
- A modern web browser (Chrome, Firefox, or Safari)
- Node.js `{>=18.0.0}` installed
- A free account on the Solarpunk dashboard
- Terminal or command prompt access
</Callout>

## Create Your Account and Access Dashboard

Follow these steps to register and log in to your Solarpunk dashboard.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.example.com/signup` and create your account using your email.
    
    No credit card required for the free tier.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification email from Solarpunk and click the link.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to `https://dashboard.example.com/login`, enter your credentials, and access the Ubik dashboard.
  </Step>
</Steps>

## Set Up Your Local Environment

Install the Solarpunk CLI and configure your API key across platforms.

<Tabs>
  <Tab title="macOS" icon="apple">
    
    Use Homebrew to install the CLI.
    
````bash
brew install solarpunk/tap/solarpunk-cli
````
    
    Configure your API key:
    
````bash
solarpunk config set api-key YOUR_API_KEY
````
  </Tab>
  <Tab title="Windows" icon="monitor">
    
    Use winget for installation.
    
````powershell
winget install Solarpunk.CLI
````
    
    Set your API key:
    
````powershell
solarpunk config:set api-key YOUR_API_KEY
````
  </Tab>
  <Tab title="Linux" icon="linux">
    
    Download and install via curl.
    
````bash
curl -sSL https://get.solarpunk.technology | bash
````
    
    Configure:
    
````bash
solarpunk config set api-key YOUR_API_KEY
````
  </Tab>
</Tabs>

<Callout kind="tip">
Obtain your API key from the dashboard under **Account > API Keys**. Replace `YOUR_API_KEY` with the actual value.
</Callout>

## Launch Your First Agentic Workflow

Create and run a simple agent that analyzes market data.

<Steps>
  <Step title="Create Agent Config" icon="file-plus">
    Create a `agent.yaml` file:
    
````yaml
name: market-analyzer
model: gpt-4o-mini
tools:
  - web-search
prompt: Analyze solar energy trends for 2024.
````
  </Step>
  <Step title="Run the Agent" icon="play">
    
    Execute in your terminal:
    
````bash
solarpunk agent run agent.yaml
````
    
    Monitor progress in the Ubik dashboard at `https://dashboard.example.com/agents`.
  </Step>
  <Step title="View Results" icon="eye">
    Check the analytics tab for decisions, actions, and proofs.
  </Step>
</Steps>
