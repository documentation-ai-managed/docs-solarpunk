These features eliminate decision fatigue by handling repetitive tasks intelligently.

## Local-First Data Processing

Keep your data secure with on-device processing. Ubik runs local models alongside your API keys, avoiding cloud training or data leaks.

<CodeGroup tabs="Node.js,Python">
  ```javascript
  const { UbikAgent } = require('@solarpunk/ubik');

  const agent = new UbikAgent({
    model: 'local-llama3',
    apiKey: 'YOUR_API_KEY',
    localOnly: true
  });

  agent.processLocalData({ input: 'user clicks' });
  ```
  ```python
  from ubik import UbikAgent

  agent = UbikAgent(
      model="local-llama3",
      api_key="YOUR_API_KEY",
      local_only=True
  )

  agent.process_local_data(input_data="user clicks")
  ```
</CodeGroup>
